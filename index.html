The Opus Audio Codec

The Opus codec is designed for interactive speech and audio transmission over the Internet, and developed as a standard by the IETF Network Working Group.

To start, you might read the latest Internet Draft or check out the prototype code: 

$ git clone git://git.xiph.org/users/jm/ietfcodec.git
